# 底部导航图标文件清单

## 文件夹路径
`宁夏文旅小程序/images/tabbar/`

## 需要复制的文件（共8个）

### 1. 首页图标
```
images/tabbar/home.png (81x81px) - 未选中
images/tabbar/home-active.png (81x81px) - 选中
```

### 2. 景点图标
```
images/tabbar/attractions.png (81x81px) - 未选中
images/tabbar/attractions-active.png (81x81px) - 选中
```

### 3. 优惠图标
```
images/tabbar/coupon.png (81x81px) - 未选中
images/tabbar/coupon-active.png (81x81px) - 选中
```

### 4. 我的图标
```
images/tabbar/profile.png (81x81px) - 未选中
images/tabbar/profile-active.png (81x81px) - 选中
```

## 配色要求
- 未选中颜色：#8B7D6B（灰褐色）
- 选中颜色：#B8860B（深金色）

## 复制步骤
1. 找到准备好的 8 个图标文件
2. 重命名为上面的文件名
3. 直接复制到 `宁夏文旅小程序/images/tabbar/` 文件夹
4. 确保文件名完全一致（区分大小写）

## 完成后
保存文件后，在微信开发者工具中点击「编译」即可看到底部导航栏。
