# 底部导航图标说明

## 图标尺寸要求
- 普通图标：81 x 81 px
- 选中图标：81 x 81 px
- 格式：PNG
- 颜色建议：
  - 未选中：#8B7D6B（灰褐色）
  - 选中：#B8860B（深金色）

## 需要的图标文件

### 1. 首页
- `images/tabbar/home.png` - 未选中状态
- `images/tabbar/home-active.png` - 选中状态
- 图标建议：房子、首页图标

### 2. 景点
- `images/tabbar/attractions.png` - 未选中状态
- `images/tabbar/attractions-active.png` - 选中状态
- 图标建议：山峰、景点图标

### 3. 优惠
- `images/tabbar/coupon.png` - 未选中状态
- `images/tabbar/coupon-active.png` - 选中状态
- 图标建议：优惠券、礼物图标

### 4. 我的
- `images/tabbar/profile.png` - 未选中状态
- `images/tabbar/profile-active.png` - 选中状态
- 图标建议：用户、个人中心图标

## 临时解决方案

如果暂时没有图标，可以使用以下在线图标资源：

### 图标网站推荐
- [Iconfont](https://www.iconfont.cn/)
- [Flaticon](https://www.flaticon.com/)
- [IconPark](https://iconpark.oceanengine.com/)

### 使用提示
1. 下载图标时选择 81x81 px 或以上尺寸
2. 使用图片处理工具调整为 81x81 px
3. 确保背景透明（PNG格式）
4. 按照颜色要求调整图标颜色
