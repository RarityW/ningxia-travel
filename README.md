# 宁夏文旅小程序

## 项目介绍

本项目是一个关于宁夏文化和旅游的小程序，旨在展示宁夏的特色景点、美食、文化等内容，主要服务人群：想购买宁夏的非遗文化产品的人群，想通过我们官方小程序来进行售卖产品的产品商

## 项目特点

- 展示宁夏特色景点
- 介绍宁夏美食文化
- 提供旅游攻略指南

## 开发环境

- 微信小程序开发工具

## 项目模块

- 美食
- 酒店
- 文创
- 出行
- 景点


## 项目结构

```
├── pages/          # 页面文件
├── components/     # 组件文件
├── utils/          # 工具函数
├── images/         # 图片资源
└── app.js          # 小程序入口文件
```
