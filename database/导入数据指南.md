# 数据导入指南

## 第一步：打开云开发控制台

1. 在微信开发者工具顶部，点击 **"云开发"** 按钮
2. 进入云开发控制台

## 第二步：创建数据库集合

在云开发控制台的 **"数据库"** 标签页：

### 创建 attractions 集合
1. 点击 **"+ 添加集合"**
2. 集合名称输入：`attractions`
3. 权限选择：**"所有用户可读，仅创建者可写"**
4. 点击确定

### 创建 food 集合
1. 点击 **"+ 添加集合"**
2. 集合名称输入：`food`
3. 权限选择：**"所有用户可读，仅创建者可写"**
4. 点击确定

### 创建 culture 集合
1. 点击 **"+ 添加集合"**
2. 集合名称输入：`culture`
3. 权限选择：**"所有用户可读，仅创建者可写"**
4. 点击确定

## 第三步：导入数据

### 导入景点数据
1. 点击 `attractions` 集合进入
2. 点击右上角的 **"导入"** 按钮
3. 在弹出的对话框中：
   - 文件格式选择：**JSON**
   - 点击 **"选择文件"**
   - 找到并选择：`database/import-attractions.json`
4. 点击 **"确定"** 开始导入
5. 等待导入完成（会显示导入成功，导入8条数据）

### 导入美食数据
1. 点击左侧的 `food` 集合进入
2. 点击右上角的 **"导入"** 按钮
3. 选择 **JSON** 格式
4. 选择文件：`database/import-food.json`
5. 点击 **"确定"**
6. 等待导入完成（导入6条数据）

### 导入文化数据
1. 点击左侧的 `culture` 集合进入
2. 点击右上角的 **"导入"** 按钮
3. 选择 **JSON** 格式
4. 选择文件：`database/import-culture.json`
5. 点击 **"确定"**
6. 等待导入完成（导入8条数据）

## 第四步：验证数据

导入完成后，你应该能在数据库控制台看到：
- `attractions` 集合：8条数据
- `food` 集合：6条数据
- `culture` 集合：8条数据

## 第五步：测试小程序

数据导入成功后，回到微信开发者工具：
1. 刷新模拟器
2. 查看首页是否显示内容
3. 由于暂时移除了底部tabBar，可以通过首页的快捷入口或直接访问页面来测试

---

## 注意事项

1. **图片链接**：目前使用的是占位图片（placeholder.com），可以正常显示
2. **权限设置**：一定要选择"所有用户可读"，否则小程序无法读取数据
3. **导入失败**：如果导入失败，请检查JSON文件格式是否正确

## 常见问题

**Q: 导入时提示"文件格式错误"怎么办？**
A: 确保选择的是 JSON 格式，且文件路径正确

**Q: 导入成功但小程序看不到数据？**
A: 检查数据库集合权限是否设置为"所有用户可读"

**Q: 图片无法显示？**
A: 检查手机/模拟器网络连接，placeholder.com 需要网络访问
